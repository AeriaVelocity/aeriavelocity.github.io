About this website

I made this website because I wanted a website for myself. I've made quite a few websites, but I've only just started doing them from scratch.

My last two attempts of making myself a website were all done in website builders.

This website is mostly just text, formatted for easy reading.

Paragraphs have line spacing, headings are tighter. Content is positioned in the middle of the browser window.

If your browser detected dark mode, you'll see a dark mode. Otherwise, it'll fallback to the default light mode. If you're using Dark Reader, set the theme generation mode to Static.

If you want to use this page's standard CSS for your own website, insert this somewhere in the document, preferably in <head>:

<link rel="stylesheet" href="https://that1m8head.github.io/standard.css">

If you do use it, please link back to this website (though you don't have to)

You can view the GitHub repo for this website. This website contains no ads or tracking. It's hosted on GitHub Pages.
