About this website

I made this website because I wanted a website for myself.
I've made quite a few websites, but I've only just started doing them using just HTML and CSS (yep, this website has no JavaScript); my last three attempts of making myself a website were all done in website builders.

This website is intentionally simplistic -- because it's part of my design philosophy for websites.
I believe the number one thing that websites should be able to do is communicate to the user through text.
Another thing in my web design philosophy is that a website should detect and apply a dark mode through CSS.

At the same time, I don't like using simple black and white -- for light mode, it's usually a warm colour and for dark mode, it's usually a dark blue.
For this website that means #333444 on #e9e4d2 and #ffffff on #171b25 for light and dark mode respectively.
