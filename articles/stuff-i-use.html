Stuff that I use

This is not a comprehensive list -- this is just my most-used stuff.

Operating system

Arch Linux, running on an old Sony VAIO PCG-61712M.

Windows 10, macOS Monterey and Linux Mint, running on a late 2013 27-inch iMac (which is considered "vintage", so suck it Apple).

No, I'm not ever going to "upgrade" to Windows 11.

Terminal

kitty is my current terminal emulator of choice.

Desktop (WMs included)

awesome and dwm as far as window managers go. I used to use i3-gaps.

Cinnamon, Xfce and Plasma for full-blown desktop environments.

Text editor

GNU Emacs with Spacemacs, usually with evil-mode.

SLADE's built-in text editor whenever I'm working on Sonic: Lock & Load.

VSCodium whenever I'm on Windows since Emacs takes ten flipping years to load. VSCodium is like VS Code but more free.

Vim or Neovim whenever I'm working at the terminal.

Browser

Brave, but I don't use any of the crypto stuff. I used to use Firefox.

surf or qutebrowser whenever I feel like using a minimal browser.

Multimedia

mpv for video and audio. It works really well and is super simple.

VLC for whenever I don't feel like using mpv.

Image editor

GIMP. It works really well.
